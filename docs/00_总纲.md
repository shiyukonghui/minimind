# MiniMind 学习教程 — 总纲

> "大道至简" — 从零开始，亲手训练一个超小语言模型

## 一、教程简介

本教程基于 [MiniMind](https://github.com/jingyaogong/minimind) 开源项目，旨在帮助 LLM 初学者从零开始理解大语言模型的每一个核心环节。MiniMind 最小版本仅 25.8M 参数（GPT-3 的 1/7000），最低只需 3 块钱服务器成本 + 2 小时即可完成从预训练到对话的全流程训练。

本教程将 MiniMind 的完整训练流程拆解为 **9 个 Phase**，由浅入深、循序渐进，每个 Phase 聚焦一个核心主题，包含原理讲解、代码解读和动手实践。

## 七、推荐学习方案

### 方案一：极速体验（2小时 + 3块钱）

适合：只想快速跑通全流程的学习者

1. Phase 1 → 搭建环境，下载已有模型体验推理
2. Phase 4 → 用 `pretrain_hq.jsonl` 预训练
3. Phase 5 → 用 `sft_mini_512.jsonl` 微调
4. 测试自己训练的模型

### 方案二：完整学习（推荐）

适合：希望深入理解 LLM 每个环节的学习者

按 Phase 1 → 9 顺序完整学习，每个 Phase 都动手实践。

### 方案三：专项深入

适合：已有基础，想深入某个方向的学习者

- 想理解模型结构 → 重点学习 Phase 3
- 想理解 RL 训练 → 重点学习 Phase 8
- 想做垂域模型 → 重点学习 Phase 6
- 想做推理模型 → 重点学习 Phase 9

## 八、前置知识要求

- **Python 基础**：熟悉 Python 语法和常用库
- **PyTorch 基础**：了解张量操作、自动求导、模型训练基本流程
- **深度学习基础**：了解神经网络、损失函数、优化器等概念
- **Linux 基础**：能够使用命令行操作（可选，Windows 也可）
